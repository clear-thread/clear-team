# Tech Stack: Clear-Team Extension

## Extension Platform
- **Gemini CLI Extension Framework**: Developed as a Gemini CLI extension for seamless integration into the developer workflow.

## Language & Runtime
- **TypeScript**: Ensuring type safety and maintainability.
- **Node.js**: Standard runtime for Gemini CLI extensions.

## Tooling & Libraries
- **Gemini CLI SDK**: For defining tools and interacting with the Gemini agent.
- **pnpm**: (Preference) Default package manager for the project.
- **Vitest**: For unit testing logic related to evolutionary psychology and game theory models.
- **Prettier & ESLint**: For code quality and consistency.

## Design & UX
- **Figma**: Primary tool for defining the product's "Phenotype" and visual identity.
- **Reactive Frameworks**: Utilizing React (Web), Compose Multiplatform (Mobile), and Flutter to build high-fidelity, interactive "Morphology."
- **nanobanana**: AI-driven image generation for rapid prototyping of "Phenotypes."
- **chrome-devtools-mcp**: MCP-based real-time UI inspection and refinement for high-fidelity interaction.

## Process & Management
- **Conductor**: Used for project planning, tracking, and specification management.
- **Context7**: Primary tool for retrieving up-to-date documentation and code examples for any library or API. Essential for resolving library IDs and querying official documentation.
